# Web To Do List Interactive

Memperkenalkan Website ini adalah web To Do list yang dimana orang bisa menambah tugas-tugas mereka dan update tugas mereka. Banyak tools yang bisa di pakai, mau itu dari tools Important, bisa select tugas Sesuai dengan hari/tanggal yang telah di input, yang memudahkan user dapat mencari tugas.

## Styling

Website ini banyak memakai style yang modern mau dari, style box yang kekinian dan juga memakai Liquid Glass Effect pada nav menu diatas. Menambahkan ada menu setting di pojok kiri bawah, yang dimana user dapan mengakses beberapa fitur pada menu widget-setting.


## Scripting

Pada script .js sudah menambahkan elemen tugas baru ke daftar.
Mengubah tampilan tugas yang sudah selesai (misalnya teks dicoret atau warna berbeda).
Menghapus tugas dari DOM saat tombol hapus ditekan.
